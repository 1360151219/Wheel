# vue-drag 可拖拽可缩放可视化弹窗

## 2022.1.26

实现基本功能：拖拽，四角缩放

参数方面：

|  参数名称  | 参数类型 |   是否必选   |         描述信息          |
| :--------: | :------: | :----------: | :-----------------------: |
| propClass  |  string  |      否      | 支持使用者自定义类名样式  |
| crashLimit |  number  | 否 (默认 10) |  四个角的最小可接触距离   |
|   title    |  string  |      否      |         弹窗标题          |
|  content   |  string  |      否      |         弹窗内容          |
|   isShow   | boolean  |      否      |       控制弹窗显示        |
|  timeout   |  number  |      否      | 让弹窗 timeout 时间后消失 |
